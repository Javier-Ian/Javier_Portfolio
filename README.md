# Ian Dave L. Javier — Creative Portfolio

A responsive one-page portfolio for an aspiring IT professional and tech creative, built with Vite and ready for Vercel.

## Customize your media

- The home-page hero uses `public/assets/ian pic.png` as its portrait.
- Save your showreel as `public/assets/showreel.mp4`. Until then, the player uses a remote sample video.
- Replace the email address and Instagram handle near the bottom of `src/main.js`.
- Replace the sample project artwork and descriptions with your real work.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

Import the repository in Vercel. It will detect Vite automatically and use `npm run build` with `dist` as the output directory.
